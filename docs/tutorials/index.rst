Overview
--------

The tutorials are organized to help you progressively build your understanding of continual learning with Mammoth Lite:

1. **First Continual Learning Experiment** - Your first complete continual learning setup
2. **Training on Custom Data** - How to adapt Mammoth Lite to your own datasets
3. **Comparing Strategies** - Systematic comparison of different continual learning approaches
4. **Advanced Configuration** - Fine-tuning hyperparameters and advanced settings

Prerequisites
-------------

Before starting the tutorials, make sure you have:

- Completed the :doc:`../getting_started` guide
- Understood the :doc:`../core_concepts` 
- Familiarized yourself with the :doc:`../examples/index`

Tutorial Format
---------------

Each tutorial includes:

- **Learning objectives** - What you'll accomplish
- **Prerequisites** - Required knowledge and setup
- **Step-by-step instructions** - Detailed walkthrough
- **Code examples** - Complete, runnable code
- **Expected results** - What outputs to expect
- **Troubleshooting** - Common issues and solutions
- **Next steps** - How to build on what you've learned

Getting Help
------------

If you encounter issues while following the tutorials:

1. Check the **Troubleshooting** section in each tutorial
2. Ensure your environment matches the :doc:`../installation` requirements
3. Open an issue on the `GitHub repository <https://github.com/aimagelab/mammoth>`_
